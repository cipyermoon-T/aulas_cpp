Trocar valores: Escreva uma função que recebe dois ponteiros para inteiros e troca os valores entre as variáveis apontadas.

Soma de array: Escreva uma função que recebe um ponteiro para um array de inteiros e o tamanho do array. A função deve retornar a soma de todos os elementos do array.

Verificar número máximo: Escreva uma função que recebe um ponteiro para um array de inteiros e o tamanho do array. A função deve encontrar e retornar o maior número do array.

Ordenar array: Escreva uma função que recebe um ponteiro para um array de inteiros e o tamanho do array. A função deve ordenar os elementos do array em ordem crescente.

Inverter string: Escreva uma função que recebe um ponteiro para uma string e inverte a ordem dos caracteres na string.

Calcular média e desvio padrão: Escreva uma função que recebe um ponteiro para um array de números reais e o tamanho do array. A função deve calcular e retornar a média e o desvio padrão dos números.
